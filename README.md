# 家校签收通

`家校签收通` 是一个面向学校的家长通知签收系统，目标是把纸质告知书、安全承诺书从“学生带回家签字”升级为“学校统一发布、家长在线确认、系统自动统计、班主任只处理异常”。

## 当前阶段

- 状态：正式开发（v0.8）
- 第一版方向：部署到服务器的网页 MVP，给真实班级试点
- 核心约束：
  - 平台为学校开通学校空间，教师账号绑定在学校下面。
  - 班主任账号和初始密码由学校统一创建，并通过老师群或指定工作群告知。
  - 班主任首次登录不强制修改初始密码。
  - 各班学生名单 Excel 由班主任导入，模板只包含学生姓名和班内序号。
  - 家长端不做账号、手机号验证码、公众号强绑定和个人中心。
  - 家长通过通知签收链接自助绑定并签收；不单独做班级冷启动绑定链接，一名学生允许绑定多个监护人。
  - MVP 支持内置和自定义通知类型，但所有通知仍要求手写签名。
  - 通知正文支持粘贴文字或上传 PDF 原件；带签名证据 PDF 由系统生成，支持单份生成和班级批量下载。
  - 班级批量 PDF 下载生成 zip 包。
  - MVP 不做全校大盘看板，只做班级进度表和导出入口。
- 部署目标域名请按实际环境配置；真实试点必须使用域名 + HTTPS。
- SSH 私钥、密码、数据库密钥等敏感信息不写入项目文档或代码。

## 正式前端

- 正式前端工程位于 `web`，采用 Next/React。
- 使用入口为 `/`：普通访问显示学校管理员/班主任登录；带 `?t=` 或 `?token=` 的家长签收链接进入家长 token 解析、绑定和签收链路。
- 运维后台入口为 `/ops`。
- 旧原型目录已移除，内部评审总览页不再作为运行时代码保留。
- 当前已覆盖：运维后台、登录、学校管理员端、学校端通知详情、班主任端、家长链接解析、家长绑定、家长签收、PDF 签收证据。
- 运维后台已接入本地后端账号：进入 `/ops` 先显示运维登录，登录成功后才展示学校空间列表、开通最小学校空间与首个学校管理员账号、停用/恢复学校空间、重置首个学校管理员密码，并展示一次性密码响应。
- 登录页已接入本地后端账号：学校管理员和班主任登录成功后会保存会话并进入对应端。
- 学校管理员端学校设置已接入本地后端：可读取学校/年级/班级/账号数据，保存学校名称，新增年级并同步创建初始班级，删除年级并清理下属班级数据，新增班级，编辑班级所属年级、名称和预计人数，新增教师账号，随机或自定义重置密码并启停账号；新增弹窗支持年级、班级、教师账号 CSV 模板下载、文件预检和确认导入，教师账号批量导入可填写可选初始密码列。
- 学校管理员端通知发布已接入本地后端：可读取通知列表，选择内置或自定义通知类型，使用文字正文或 PDF 附件，存为草稿，发布草稿，或新建后直接发布并生成签收任务。
- 学校管理员端导出归档已接入本地后端：可按通知/班级创建 Excel、单份 PDF、班级 PDF zip 任务，并通过鉴权接口下载。
- 学校管理员端班级进度和通知详情已接入本地后端：进度表可读取后端概览；后端成功返回空结果时显示空状态，不再回退到演示班级进度；可查看通知详情，修改截止时间，筛选班级未签学生，并带通知/班级范围跳转导出归档。
- 班主任端学生导入向导已接入本地后端：登录后读取后端账号分配的学校/班级上下文，模板下载、文件预检、确认导入和最近导入历史只使用本班 classId。
- 班主任端通知工作台已接入本地后端：通知列表、今日待办、签收链接重新生成/撤销、转发标记、进度刷新、学生签收详情、提醒记录、本班异常确认/驳回和本班导出任务创建/下载可直接调用 API；今日待办由本班通知投递、签收进度和异常状态自动生成。
- 家长端已接入公开 token API：链接解析读取真实 token 范围，绑定页提交学生/监护人/签名并保存本机绑定上下文，签收页可查看 PDF 通知附件，提交阅读确认、隐私确认和手写签名并生成回执。
- 视觉定位为“学校教务台账”：克制、密集、可扫读，适合班主任和学校管理员反复处理。
- 旧界面稿中的关键产品结论已经吸收到当前前端实现和公开说明中；完整内部 PRD 不随开源仓库发布。

## 后端开发进展（MVP 骨架）

- 新增 `server` 目录，包含：
  - `server/package.json`
  - `server/server.js`
- 当前服务已覆盖：
  - 基础鉴权（平台运维/学校管理员/班主任）与 demo 快速通道（本地开发可用 `x-demo-user` 头）
  - 平台运维开通学校空间、首个学校管理员账号、停用/恢复学校空间、重置首个学校管理员密码
  - 学校管理员维护学校名称、班级、教师账号、账号重置和停用
  - 学生导入模板下载、Excel/CSV 文件预检、粘贴预检、提交与导入历史查询
  - 通知班级投递记录、转发标记、提醒记录
  - 班主任当前用户上下文（学校、班级、班主任、本班学生数）
  - 通知签收链接签发、撤销和重发；未绑定家长可使用同一个通知链接先绑定再签收
  - 家长绑定、绑定异常落库、家长签收、重复签收拦截、签收过期记录
  - 通知发布、生成签收任务
  - 班级进度聚合与异常状态基础口径
  - 导出任务（Excel 兼容 `.xls`、单份 PDF、班级 PDF zip）与下载鉴权
  - 可选账号密码修改（`/api/user/change-password`）
  - 文件落盘：`server/data/uploads`、`server/data/exports`


启动后可访问：

- 正式前端：http://localhost:3000
- 运维后台：http://localhost:3000/ops
- 后端根地址：http://localhost:8088
- 健康检查：http://localhost:8088/api/health
- 本地测试账号：

  - `x-demo-user: teacher` / `x-demo-user: school_admin` 仅在已显式种入学校 demo 数据时可用于开发调试。

## 环境变量

| 变量 | 用途 | 默认值 |
|---|---|---|
| `PORT` | 后端监听端口 | `8088` |
| `PUBLIC_APP_BASE_URL` | 后端生成家长签收链接时使用的前端地址 | `http://localhost:3000` |
| `OPS_ADMIN_USERNAME` | 首次初始化平台运维账号用户名 | `platform_admin` |
| `OPS_ADMIN_PASSWORD` | 首次初始化平台运维账号密码；留空则生成一次性随机密码 | 留空 |
| `PDF_FONT_PATH` | 中文 PDF 字体路径 | 自动尝试常见字体 |
| `CORS_ORIGIN` | 允许跨域访问 API 的前端域名，多个用逗号分隔 | 留空时使用开发默认 |
| `JIAXIAO_SEED_DEMO` | 设置为 `1` 时写入本地演示数据 | 留空 |
| `NEXT_PUBLIC_API_BASE_URL` | 前端调用的后端 API 地址 | `http://localhost:8088` |

生产部署前至少设置 `PUBLIC_APP_BASE_URL`、`NEXT_PUBLIC_API_BASE_URL`、`CORS_ORIGIN`、`OPS_ADMIN_PASSWORD` 和 `PDF_FONT_PATH`。
